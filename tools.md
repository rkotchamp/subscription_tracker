```markdown
# Tools Used in the App

## Frontend

- **Next.js**: Framework for building the UI and managing server-side rendering.
- **React**: For creating dynamic, reusable components.
- **ShadCN UI**: Pre-designed components for faster development.
- **TailwindCSS**: Utility-first framework for styling.

## Backend

- **Next.js**: Provides the server-side logic.
- **MongoDB**: Stores user, subscription, and notification data.

## Authentication

- **JWT**: Manages user authentication.
- **OAuth2**: Securely connects email accounts.

## Email Parsing

- **Nodemailer**: Fetches and processes email data.
- **Zod**: Validates parsed email data to ensure accuracy.

## Notifications

- **Firebase Cloud Messaging**: Sends push notifications.
- **Twilio**: Provides SMS and email notifications as an alternative.

## File Storage

- **Firebase Storage**: Handles uploaded invoices securely.

## Deployment

- **Vercel**: For hosting the application.
- **CI/CD Pipelines**: Streamline updates and releases.
```
